# Project: NodeBase Workflow Platform

**Description:**
NodeBase Workflow is a modern, full-stack automation platform built to streamline complex business processes through a visual, drag-and-drop interface. It enables users to design, execute, and monitor automated workflows that integrate various services and API endpoints without writing extensive code.

**Key Technologies:**
- **Frontend:** Next.js 16, TypeScript, React Flow, Tailwind CSS v4, Shadcn UI.
- **Backend:** Node.js, Inngest (Event-driven architecture), Prisma ORM, PostgreSQL.
- **Auth & Security:** Better-Auth, Secure Credential Management.
- **AI:** Integrated AI SDKs for intelligent automation nodes.

**Highlights:**
- Developed a scalable visual workflow editor allowing for conditional logic, loops, and external integrations.
- Implemented a robust background job processing system using Inngest to ensure reliable execution of long-running tasks.
- Designed a secure architecture for handling sensitive user credentials and API keys.
- Built a responsive and accessible dashboard for real-time monitoring of workflow executions and logs.
